# termsweeper
A clone of Minesweeper, only using standart C Libraries. Currently very WIP!

## Building
### Dependencies:
```
git
build-essentials
```

1. Clone the repository `git clone https://github.com/LukaCPP/termsweeper.git`

2. Change into the directory you cloned the repo to

3. `make dirs`
4. `make`
